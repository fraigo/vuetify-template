## Setup a new project

* `vue create vue-template`
    * Use vue-2 template
* `cd vue-template`
* `vue add vuetify`
    * Use Preset: Vuetify 2 - Vue CLI (recommended)
* `vue add router`
    * Use history mode for router? (Requires proper server setup for index fallback in production) No
* `vue add vuex`
* `npm install -D sass-loader node-sass`
* `vue add pwa`
